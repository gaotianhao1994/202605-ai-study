# 202605-AI-study
Repository for AI study and learning

## 项目简介

这是一个用于 AI 学习和研究的项目仓库，包含各种 AI 相关的演示程序和实验代码。

## 目录结构

- `20260507-demo/` - 第一个演示项目
- `20260507-demo02-thinking-reflection-framework/` - 思维反思框架演示项目

## 环境要求

- Python 3.x

## 安装和配置

1. 克隆仓库：
   ```bash
   git clone https://github.com/gaotianhao1994/202605-ai-study.git
   cd 202605-ai-study
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用示例

### 思维反思框架演示

进入演示目录并运行：

```bash
cd 20260507-demo02-thinking-reflection-framework
python demo.py
```
